<h1>鳶尾花資料集
  <h2>資料集介紹及目標  
  鳶尾花資料集內總共有四個特徵，三種分類結果  
  目標是利用這四個特徵，完成三種鳶尾花的分類問題  
 
<h3>iris dataset資料集為  

* SepalLengthCm 花萼長度
* SepalWidthCm 花萼寬度
* PetalLengthCm 花瓣長度
* PetalWidthCm 花瓣寬度
* Species 品種
  * Iris-setosa
  * Iris-versicolor
  * Iris-virginica
